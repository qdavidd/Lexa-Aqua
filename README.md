# HYDROSAFE

Real-time water quality alerts for irrigation farmers in Romania.

Demo for CASSINI Hackathon 11 - Space for Water (24-26 April 2026).

Uses Copernicus Sentinel-3 OLCI and Sentinel-2 MSI to detect pollution events on rivers, then sends SMS alerts to farmers before they pump contaminated water onto their crops.

## Run locally

```
pip install -r requirements.txt
streamlit run streamlit_app.py
```
